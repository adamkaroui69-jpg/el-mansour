## 3. Material Design Components

### 3.1 Color Scheme

**Primary Colors**:
- Primary: Blue (#2196F3)
- Primary Dark: Dark Blue (#1976D2)
- Accent: Orange (#FF9800)

**Background**:
- Light Theme (default)
- Card Background: White
- Surface: Light Gray (#F5F5F5)

### 3.2 Typography

**Fonts**:
- Primary: Roboto (Material Design default)
- Headings: Roboto Medium
- Body: Roboto Regular

**Sizes**:
- H1: 34sp
- H2: 24sp
- H3: 20sp
- Body: 14sp
- Caption: 12sp

### 3.3 Components

**Used Components**:
- MaterialDesign:Card
- MaterialDesign:Button
- MaterialDesign:TextBox
- MaterialDesign:ComboBox
- MaterialDesign:DataGrid
- MaterialDesign:DialogHost
- MaterialDesign:SnackbarMessageQueue

## 4. Responsive Design

### 4.1 Window Sizes

**Minimum Size**: 1024Ã—768
**Recommended Size**: 1280Ã—720
**Maximum Size**: No limit

### 4.2 Layout Adaptation

- Sidebar: Fixed width (200px)
- Content area: Flexible, minimum 800px
- Cards: Responsive grid (2-3 columns)

---

## 5. User Experience Guidelines

### 5.1 Feedback

- **Success**: Green snackbar
- **Error**: Red snackbar
- **Warning**: Orange snackbar
- **Info**: Blue snackbar

### 5.2 Loading States

- Progress indicators for async operations
- Skeleton loaders for data loading
- Disable buttons during processing

### 5.3 Error Handling

- Clear error messages in French
- Validation feedback on input fields
- Retry options for failed operations

---

## 6. Accessibility

### 6.1 Keyboard Navigation

- Tab order logical
- Enter to submit forms
- Escape to close dialogs
- Arrow keys for lists

### 6.2 Screen Readers

- Proper labels on all controls
- ARIA attributes where needed
- Descriptive tooltips
